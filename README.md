# dom-inspect-overlay

Application is a webpage inspector overlay, it loads an iframe on the same domain, user can select/inspect sections of the dom, and use a form to enter labels for each section/element. DOM nodes can be labeled and categorized, stored, listed, and edited.

## Setup

```sh
git clone https://github.com/GPT-Engineer-App/dom-inspect-overlay.git
cd dom-inspect-overlay
npm i
```

```sh
npm run dev
```

This will run a dev server with auto reloading and an instant preview.
